# Zen Dao

> Minimalist DAO governance platform

## Features

- Proposal creation
- Voting system
- Treasury management
- Member roles

## Tech Stack

- Frontend: HTML5, CSS3, JavaScript
- Blockchain: Solidity, Rust, Web3.js
- AI: MiMo V2 Flash, Claude Code

## Installation

```bash
git clone https://github.com/mupoiyu/zen-dao.git
cd zen-dao
npm install
npm run dev
```
